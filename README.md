# AR Wall Preview MVP

ეს არის იზოლირებული პროტოტიპი, რომელიც ძირითად აპს არ ეხება.

არსებული ფუნქციები:

- კამერის გახსნა
- 4 წერტილით კედლის მონიშვნა
- ერთი ტექსტურის ატვირთვა
- scale კონტროლი
- screenshot

ტესტისთვის გახსენი:

- `mvp-ar-preview/index.html`

კამერა საუკეთესო შედეგით იმუშავებს:

- `localhost`-ზე
- ან `https` გარემოში

შემდეგი შესაძლო ნაბიჯები:

- პროდუქტის picker
- wall width/height inputs
- უკეთესი perspective correction
- პროექტზე მიბმა / save preview
